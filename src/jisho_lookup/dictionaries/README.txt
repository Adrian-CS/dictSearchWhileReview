Coloca aquí tus ZIPs de Yomichan/Yomitan (por ejemplo jmdict_english.zip,
jitendex.zip, daijirin.zip, etc.).

El add-on los indexará al arrancar Anki y los usará como fallback si la
API de Jisho no responde.

Puedes activar/desactivar cada diccionario desde:
  Herramientas → Jisho Lookup → Configuración
